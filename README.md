# ClickBay - AI Virtual Assistant & Content Analytics

**ClickBay** is a modern, AI-powered virtual assistant system designed for content scraping, analysis, and conversion. It provides users with deep insights into web content through advanced sentiment analysis, keyword extraction, and entity recognition.

> **Note:** This project is currently in active development. The frontend is fully functional, while the backend implementation is ongoing.

## 🚀 Features

- **AI-Powered Scrape & Search:** Effortlessly scrape content from any URL and get instant AI-driven insights.
- **Advanced Analytics Dashboard:**
  - **Sentiment Analysis:** Understand the emotional tone of any content with confidence scores and detailed breakdowns.
  - **Keyword Cloud:** Visualize top keywords and trending topics.
  - **Entity Recognition:** Automatically identify key people, organizations, and locations mentioned in the text.
  - **Topic Distribution:** See how content is categorized across different domains like AI, Tech, and Business.
  - **AI Summary:** Get concise, AI-generated summaries of long-form content.
- **Personalized Experience:**
  - **User Profiles:** Secure login and account management.
  - **Notes & History:** Keep track of your highlights, hooks, and search history.
- **Connectivity Hub:** Integrate with MCP servers and external APIs (in progress).

## 🛠️ Technology Stack

- **Frontend:** HTML5, Modern CSS (Custom + Materialize), TypeScript
- **Design:** Premium UI with glassmorphism, dynamic gradients, and responsive layouts.
- **Icons & Fonts:** Material Icons, Google Fonts (Inter, Roboto, Outfit)
- **Development Tools:** Live-Server, TypeScript Compiler (`tsc`)

## ⚙️ Installation & Setup

1. **Clone the repository:**

   ```bash
   git clone https://github.com/royChuks/ClickBay.git
   cd ClickBay/frontend
   ```

2. **Install dependencies:**

   ```bash
   npm install
   ```

3. **Run in development mode:**
   ```bash
   npm run dev
   ```
   _This will start a TypeScript watch process and a live-server instance._
